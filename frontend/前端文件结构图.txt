================================================================================
  技术栈摘要
================================================================================
  框架:       Vue 3 (Composition API / <script setup>)
  构建工具:   Vite 5
  UI 组件库:  Element Plus 2.x
  状态管理:   Pinia
  路由:       Vue Router 4 (createWebHistory)
  HTTP:       Axios + 原生 fetch
  CSS:        SCSS (Element Plus 变量覆盖) + 原生 CSS (设计变量 / 全局覆盖)
  语言:       JavaScript (ES Module)

================================================================================
  设计体系
================================================================================
  风格:       日系简约·阳光活力
  主色:       暖杏 #F4A460 (linear-gradient → #F7C57C)
  辅色:       山吹黄 #F9E0A0 / 薄荷绿 #A8D8B9 / 珊瑚粉 #EFA8A8 / 天蓝灰 #A9C1D9
  背景:       米白 #F9F8F6 (页面) / 纯白 #FFFFFF (卡片)
  圆角:       基础 14px / 卡片 24px / 按钮 28px 胶囊形
  阴影:       轻柔分层 (0 2px 8px ~ 0 12px 36px)
  动画:       cubic-bezier(0.34, 1.56, 0.64, 1) 弹性曲线
  响应式:     手机 <768px (汉堡菜单+抽屉/表格横滚/表单竖排/弹窗全屏)
              平板 768-1023px (tabs保持/双列网格/弹窗85%)

================================================================================
  路由结构（router/index.js）
================================================================================

  【C端前台 — ClientLayout 包裹】
  /                              → 重定向到 /studios
  /studios                       → studio/StudioList.vue（工作室宫格列表）
  /studios/:id                   → studio/StudioDetail.vue（工作室详情）
  /styles                        → style/StyleList.vue（预设列表）
  /booking/:studioId             → client/Window_C_Step1.vue（预约步骤1）
  /booking/:studioId/step2       → client/Window_C_Step2.vue（预约步骤2）

  【B端后台 — AdminLayout 包裹】
  /admin                         → 重定向到 /admin/orders
  /admin/login                   → admin/LoginView.vue（登录/注册）
  /admin/orders                  → admin/OrderManagement.vue（订单管理）[需登录]
  /admin/styles                  → admin/StylesView.vue（预设管理）[需登录]
  /admin/styles/create           → 重定向到 /admin/styles
  /admin/studios                 → studio/StudioList.vue（项目管理）[需登录]
  /admin/studio/create/step1     → admin/Window_B_Step1.vue（创建项目-步骤1）[需登录]
  /admin/studio/create/step2     → admin/Window_B_Step2.vue（创建项目-步骤2）[需登录]
  /admin/studio/create/step3     → admin/Window_B_Step3.vue（创建项目-步骤3）[需登录]
  /admin/studio/edit/:id         → admin/EditStudio.vue（编辑项目）[需登录]
  /admin/settings                → admin/SettingsView.vue（系统设置）[需登录]
  /admin/notifications           → admin/NotificationsView.vue（通知管理）[需登录]
  /admin/logs                    → admin/LogsView.vue（操作日志）[需登录]
  *                              → NotFoundView.vue（404）

  全局导航守卫:
    - 已登录访问 /admin/login → 重定向到 /admin/orders
    - 需鉴权路由无 Token → 重定向到 /admin/login

  构建产物 (dist/)
================================================================================
